GitHub Users in Africa  

This map represents all GitHub users as of Jan 1st, 2015 who had their profile location in GitHub set to either an African country or city.

Since this ONLY counts users who set their location, this represents a best attempt to show where and to what degree GitHub is being adopted.

If you are a GitHub user living in Africa, please update your location to be counted in future versions of this map.

This is an initiative of the CodeAfrica project which seeks to grow, promote and highlight coders in Africa.

This map was made by Ona and Yeleman - two companies creating world class open source software in Nairobi, Kenya and Bamako, Mali.
